# Contributing

## Requirements

- Python 3
- [go-task](https://taskfile.dev/installation)

## Testing

To run all tests:

```bash
task test
```

## Linting

To run all linting checks:

```bash
task lint:check
```

## Other Commands

To clean up test files:

```bash
task lint:check
```

You can also run specific unit tests with `task tests:*`

A list of available tasks can be found with `task --list-all`
